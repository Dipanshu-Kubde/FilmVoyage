# 🎬 FilmVoyage

**FilmVoyage** is a modern, fast, and elegant movie and TV show discovery platform. Powered by [TMDb API](https://www.themoviedb.org/), it helps users explore trending content, top-rated gems, and timeless classics — all in one clean, responsive interface.

> 🚀 *"Embark on a cinematic journey like no other. Discover, explore, and fall in love with movies all over again."*

---

## 📌 Features

- 🔍 **Search & Filter** across thousands of titles  
- 📈 **Trending**, **Top Rated**, and **What's Popular** sections  
- 🎭 Genre-based discovery (Action, Drama, Comedy, etc.)  
- 🖼️ Lazy-loading posters and infinite scrolling for smooth UX  
- 🎥 Movie trailers with embedded video playback  
- 📅 Dynamic date formatting with Day.js  
- 🎨 Beautiful, responsive UI built with SCSS and React

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Dipanshu-Kubde/FilmVoyage.git
cd FilmVoyage
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Add TMDb API Key

Create a `.env` file in the root directory and add your TMDb API key:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key_here
```

Get your API key here: [https://www.themoviedb.org/settings/api](https://www.themoviedb.org/settings/api)

### 4. Start the Development Server

```bash
npm run dev
```

Your app should now be running at `http://localhost:5173`
